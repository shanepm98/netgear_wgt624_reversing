Bug Fixes

  1. Fixed - DHCP reboot issue by heavy traffic.
  2. Fixed - Schedule does not work across midnight.

Installation Instructions

Please perform firmware upgrade with “wired” or Ethernet connection only.

  1. Open the browser and access the router (usually the address is http://192.168.1.1).
  2. Log in to the router (User Name = admin and Password = password unless you have changed it).
  3. Click Maintenance > Router Upgrade.
  4. Click Browse and locate the upgrade file (download link is below).
  5. Click Upload.
  6. Wait for the router to reboot.
  7. Verify that the router firmware was upgraded.
  8. Reset router to factory defaults.
  9. Wait for the router to reboot.
  10. Reenter the settings you noted down before upgrading.
